# 🌐 Blockchain Journey

Welcome to my Web3 and Blockchain development repository! This repository serves as a personal ledger of my learning journey, containing core JavaScript algorithms, asynchronous simulations, and decentralized logic structures.

## 🚀 Key Modules & Simulations Implemented:

1. **MetaMask Wallet Connection (`wallet-connect.js`)**
   - Implements a simulated decentralized wallet authentication flow.
   - Utilizes asynchronous JavaScript (`async/await`) to handle simulated network states and secure handshakes.

2. **Asynchronous Transaction Processing (`transaction.js`)**
   - Simulates the lifecycle of a blockchain transaction from initiation to confirmation.
   - Handles promise-based states representing node validations.

3. **Automated Toll Fee Splitter (`app.js` & `index.html`)**
   - A fully functional frontend-integrated logic that automates fee distribution.
   - Features a core 15% automated toll fee splitting mechanism, mimicking smart contract distribution structures.

## 🛠️ Tech Stack:
- JavaScript (ES6+)
- HTML5 / CSS3 (Frontend integration)
- Asynchronous Programming (Promises, Async/Await)
